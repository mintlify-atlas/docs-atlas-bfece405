This guide walks you from a fresh clone to a running system where you register a product and complete your first rental.

## Prerequisites

- Node.js 18 or later
- A MongoDB Atlas account (free tier works)
- A Gmail account for email notifications (or skip email during initial setup)

## Step 1: Clone and install

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/StewiePro/pycasasAlquileres.git
    cd pycasasAlquileres
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Create your environment file">
    Create a `.env` file in the project root:

    ```bash .env
    MONGODB_URI=mongodb+srv://<user>:<password>@<cluster>.mongodb.net/?retryWrites=true&w=majority
    PORT=3000
    ```

    Replace the connection string with your MongoDB Atlas URI. See [MongoDB Setup](/setup/mongodb) for details.
  </Step>
</Steps>

## Step 2: Start the server

```bash
node server.js
```

You should see:

```
✉️ Nodemailer (SMTP Dreamhost) listo para producción
✅ Conectado a MongoDB
🚀 Servidor corriendo en http://localhost:3000
```

Open your browser at `http://localhost:3000`.

## Step 3: Log in with the demo account

On the login page, use the built-in demo admin credentials:

| Field | Value |
|-------|-------|
| **Username** | `Pruebas` |
| **Password** | `123` |
| **Role** | `Administrador` |

<Note>
The `Pruebas` account runs in **test mode** — all data is stored separately from production data. The header turns orange to remind you. Use `Pycasas2012` / `Pycasas2012` for the production demo account.
</Note>

## Step 4: Add your first product

<Steps>
  <Step title="Open the product entry form">
    In the left sidebar, click **Ingresar Artículo**.
  </Step>
  <Step title="Fill in product details">
    Complete the form with:
    - **Código**: A unique SKU (e.g., `ANDAM-001`)
    - **Nombre**: Product name (e.g., `Andamio Tubular 1m`)
    - **Categoría**: Select a category
    - **Precio/día**: Rental price per day
    - **Stock**: Available quantity
    - **Stock mínimo**: Low-stock alert threshold
  </Step>
  <Step title="Save the product">
    Click **Guardar Producto**. The system saves to localStorage and automatically syncs to MongoDB Atlas.
  </Step>
</Steps>

## Step 5: Create your first rental

<Steps>
  <Step title="Open rental checkout">
    In the sidebar, click **Salida por Alquiler**.
  </Step>
  <Step title="Add client information">
    Fill in the client's name, ID number, and phone number.
  </Step>
  <Step title="Search and add products">
    Use the article search box to find the product you added. Click **Agregar** and enter the quantity.
  </Step>
  <Step title="Set a guarantee amount (optional)">
    Enter a guarantee amount in the **Garantía** field. This is deducted from the total at checkout.
  </Step>
  <Step title="Register the rental">
    Click **Registrar Salida**. The system will:
    - Deduct stock from inventory
    - Assign a remission number (e.g., `REM00001`) and invoice number (e.g., `FVPA00001`)
    - Open the remission note in a new tab
    - Sync data to MongoDB
  </Step>
</Steps>

## Step 6: Process a return

<Steps>
  <Step title="Open the return module">
    Click **Devolución Alquiler** in the sidebar.
  </Step>
  <Step title="Search for the active rental">
    Search by client name, ID number, or rental ID.
  </Step>
  <Step title="Select the rental and confirm return">
    Click **Devolver**, verify the return date and days, apply any guarantee charge, then click **Confirmar Devolución**. The system restores stock and calculates the final billing.
  </Step>
</Steps>
